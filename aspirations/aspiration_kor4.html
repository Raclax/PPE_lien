위키백과, 우리 모두의 백과사전.

집합론에서 관계(關係, 영어: relation)는 곱집합의 부분 집합이다. 원소의 튜플이 이 부분 집합에 속하는지 여부를 통해 원소들 사이의 관계를 나타낸다.

정의[편집]

집합족 { X i } i ∈ I {\displaystyle \{X_{i}\}_{i\in I}} 위의 관계는 곱집합의 부분 집합

R ⊆ ∏ i ∈ I X i {\displaystyle \textstyle R\subseteq \prod _{i\in I}X_{i}}

이다. 특히, 순서수 α {\displaystyle \alpha } 에 대하여, 집합 X {\displaystyle X} 위의 α {\displaystyle \alpha } 항 관계(영어: α {\displaystyle \alpha } -ary relation)는 거듭제곱 집합의 부분 집합

R ⊆ X × α {\displaystyle R\subseteq X^{\times \alpha }}

이다. 특히 항수에 따라 다음과 같은 특수한 경우들을 생각할 수 있다.

  • X {\displaystyle X} 위의 영항 관계(零項關係, 영어: nullary relation) R ⊆ { ∙ } {\displaystyle R\subseteq \{\bullet \}} . 즉, R = ∅ {\displaystyle R=\varnothing } 또는 R = { ∙ } {\displaystyle R=\{\bullet \}} .
  • X {\displaystyle X} 위의 단항 관계(單項關係, 영어: unary relation) R ⊆ X {\displaystyle R\subseteq X}
  • X {\displaystyle X} 위의 이항 관계(二項關係, 영어: binary relation) R ⊆ X × X {\displaystyle R\subseteq X\times X} . 이 경우, ( x , y ) ∈ R {\displaystyle (x,y)\in R} 를 x R y {\displaystyle xRy} 와 같이 표기하기도 한다.
  • X {\displaystyle X} 위의 삼항 관계(三項關係, 영어: ternary relation) R ⊆ X × X × X {\displaystyle R\subseteq X\times X\times X} .
  • X {\displaystyle X} 위의 n {\displaystyle n} 항 관계(-項關係, 영어: n-ary relation) R ⊆ X × n {\displaystyle R\subseteq X^{\times n}} . 이 경우, ( x 1 , … , x n ) ∈ R {\displaystyle (x_{1},\dots ,x_{n})\in R} 를 R ( x 1 , … , x n ) {\displaystyle R(x_{1},\dots ,x_{n})} 와 같이 표기하기도 한다.

예[편집]

  • 공선점 관계는 공간 속의 점들에 대한 삼항 관계이다.
  • 여러 가지 이항 관계: 여기서는 집합 X {\displaystyle X} 상의 이항 관계 R {\displaystyle R} 에 관해서 생각하자. ∀ x ∈ X ( x R x ) {\displaystyle \forall x\in X(xRx)} 를 반사성(reflexivity)라 한다. ∀ x , y ∈ X ( x R y → y R x ) {\displaystyle \forall x,y\in X(xRy\rightarrow yRx)} 를 대칭성(symmetricity)라 한다. ∀ x , y ∈ X ( x R y ∧ y R x → x = y ) {\displaystyle \forall x,y\in X(xRy\wedge yRx\rightarrow x=y)} 를 반대칭성(antisymmetricity), ∀ x , y ∈ X ( x R y → ¬ y R x ) {\displaystyle \forall x,y\in X(xRy\rightarrow \neg yRx)} 를 비대칭성(asymmetricity)이라 한다. ∀ x , y , z ∈ X ( x R y ∧ y R z → x R z ) {\displaystyle \forall x,y,z\in X(xRy\wedge yRz\rightarrow xRz)} 를 추이성이라 한다.
  • 반사적, 대칭적, 추이적인 관계를 동치 관계라 한다.
  • 반사적, 반대칭적, 추이적인 관계를 순서 관계라 하고 비대칭적, 추이적인 관계를 강 순서 관계라 한다. 그러면 관계 < {\displaystyle <} 와 관계 ≤ {\displaystyle \leq } 에 대하여 x ≤ y ⟺ x < y ∨ x = y {\displaystyle x\leq y\iff x<y\vee x=y} 가 성립할 때, < {\displaystyle <} 가 강 순서 관계일 조건과 ≤ {\displaystyle \leq } 가 순서 관계일 조건은 서로 필요충분조건이다. 문맥에 따라 강 순서 관계를 그저 순서 관계로 할 수도 있으므로 주의가 필요하다.
